Organization: PureGym US (formerly Blink Fitness)
Date: September 2025

Project Overview

Served as a key technical lead and execution owner for the Blink to PureGym US enterprise IT migration. The initiative focused on identity consolidation, endpoint modernization, security alignment, automation, and operational readiness across both headquarters and field environments.

Identity, User & Data Preparation

  • Conducted deep audits of 2,000+ user accounts
  • Identified and migrated 315 active, validated users
  • Led cleanup of approximately 10TB of obsolete legacy data
  • Rebuilt and standardized shared mailboxes and distribution groups
  • Implemented US-prefixed naming standards across identity objects

Endpoint & Device Management Leadership

  • Designed and deployed Intune, Jamf, and Apple Business Manager configurations
  • Implemented Windows 11 Autopilot for HQ, clubs, and gate systems
  • Led Jamf pre-stage enrollment for macOS devices
  • Defined role-based enrollment profiles for iOS devices
  • Automated deployment of Cisco Umbrella, TeamViewer Host, SonicWall VPN, and Parallels Desktop
  • Led Windows 11 upgrades across hundreds of devices

Cloud, Security & Access Control

  • Supported migration of cloud storage into Microsoft 365 and SharePoint
  • Aligned Google Workspace security controls with Microsoft Defender
  • Implemented Entra ID PIM-based access and least-privilege admin roles
  • Assisted in designing E3/E5 and F3/F5 licensing strategy

Automation & Engineering

  • Developed PowerShell scripts to automate M365 object creation
  • Automated license assignment and SharePoint access management
  • Built dynamic device groups and enrollment profiles

Operational Readiness & Support

  • Defined US IT admin support model with secure elevated access
  • Coordinated with vendors on Intune and Apple Business Manager enrollment
  • Ensured smooth go-live execution with no critical disruptions

Technologies

  • Microsoft 365 & Entra ID
  • Intune, Jamf & Apple Business Manager
  • Windows 11, macOS, iOS
  • Microsoft Defender & Azure
  • PowerShell Automation
