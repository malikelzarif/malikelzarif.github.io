Project Overview

Served as a key technical lead and execution owner for the Blink to PureGym US enterprise IT migration, focusing on identity, endpoint management, automation, security alignment, and operational readiness across HQ and field environments.

Identity, User & Data Preparation

  • Conducted deep audits of 2,000+ user accounts, identifying 315 valid users for migration
  • Led cleanup of ~10TB of obsolete data from legacy and former employee accounts
  • Rebuilt and standardized distribution groups and shared mailboxes
  • Implemented US-prefixed naming standards for users, mailboxes, and groups

Endpoint & Device Management Leadership

  • Designed and deployed Intune, Jamf, and Apple Business Manager configurations for US operations
  • Implemented Windows 11 Autopilot for HQ, Clubs, and Gate systems
  • Implemented Jamf pre-stage enrollment for macOS devices
  • Implemented role-based enrollment profiles for iOS devices
  • Automated deployment of core applications including Cisco Umbrella, TeamViewer Host, SonicWall VPN, and Parallels Desktop
  • Led Windows 11 upgrades across hundreds of devices prior to migration

Cloud, Security & Access Control

  • Supported migration of cloud storage into Microsoft 365 / SharePoint
  • Aligned Google Workspace security controls with Microsoft Defender
  • Implemented Entra ID PIM-based access and least-privilege admin roles
  • Assisted in designing E3/E5 and F3/F5 licensing strategy

Automation & Engineering

  • Developed PowerShell scripts to automate M365 object creation
  • Developed PowerShell scripts to automate license assignment
  • Developed PowerShell scripts to automate SharePoint access management
  • Built dynamic device groups and enrollment profiles to support scalable device onboarding

Operational Readiness & Support

  • Helped define the US IT admin support model with secure elevated access
  • Coordinated with vendors on Intune and Apple Business Manager enrollment
  • Ensured smooth go-live execution with no critical operational disruptions

Technologies
